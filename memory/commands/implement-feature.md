# /implement-feature

Use the implementation engineer and architecture guidance to deliver a scoped feature safely.

## Steps

1. Inspect relevant modules.
2. Read applicable policies and rules.
3. Implement smallest safe diff.
4. Run targeted tests.
5. Run code-smell audit.
6. Update docs if behavior changed.

## Recommended delegates

- `java-implementation-engineer`
- `code-smell-auditor`
- `technical-writer` when needed
